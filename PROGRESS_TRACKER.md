# Progress Tracker

  *Completion Set*

  _End State_

  Goal: Submit response to request of information <= deadline that increases confidence in hypothesis that client will win case.

  Requirements:
  - 20 responses to 20 requests
  - Discrete response to each request
  - Delivered as 1 zip file per request

  User can in 5 step process only taking 1 minutes per step (5 minutes total +/- 20% tolerance [implied standard deviation of set of outcomes] @ 95% confidence) assemble the set of data from the assumed data set of the case to answer an information request (1/20th) of the 20 defining the current state in the current regime of the case.

  _Intermediate State_

Goal: Application helps attorney complete the information request in < 2 hours.

Requirements:
- Attorney can download application from poincareresearch.com
- Application is a wails desktop application that runs on Windows, Mac, and Linux
- Data Lake is accessible via API from web application orchestrating Google Cloud 100 GB instance
